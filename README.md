# Gameboy DMG Assembly Boilerplate

Some boilerplate assembly to start a new DMG project.

Includes some files from [hardware.inc](git@github.com:gbdev/hardware.inc.git) as a submodule. Be sure to clone with:
```bash
git clone --recurse-submodules git@github.com:pgalasti/dmg-test.git
```

or

```bash
git submodule update --init --recursive
```

after cloning.
